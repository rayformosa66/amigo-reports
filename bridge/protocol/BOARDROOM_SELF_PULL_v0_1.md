# BOARD ROOM SELF-PULL v0.1

Purpose:
Standing protocol so Wolfe can pull agenda state from the Board Room Window and Inbox without Ray carrying message cargo.

Rules:
- Wolfe reads bridge/status/BOARDROOM_WINDOW.md
- Wolfe reads bridge/mailbox/BOARDROOM_INBOX_INDEX.md
- Wolfe may propose the next public-safe agenda item.
- Wolfe may not execute work unless AND-gate passes or Chair explicitly rules.
- If unclear, Wolfe HOLDS and asks for Chair ruling.
- No private cargo.
- No deploy, deletes, secrets, paid API, permission changes, or broad approvals.
